# Card Matching Game

A simple card matching game built with plain HTML, CSS, and JavaScript.

## How to Play

- Click on cards to flip them and reveal the words.
- Find pairs of words with similar meanings.
- If two flipped cards match, they stay uncovered.
- If not, they flip back after a short delay.
- Match all pairs to win!

## Words

1. 飯盒 (rice box) = 盒飯 (lunch box)
2. 睇波 (watch ball) = 看球 (watch soccer)
3. 魚蛋 (fish egg) = 魚丸子 (fish ball)
4. 起樓 (build building) = 蓋房子 (build house)

## Running the Game

Open `index.html` in a web browser to start the game.

## Testing

The project uses Playwright for end-to-end testing.

To run tests:

1. Install dependencies: `npm install`
2. Install Playwright browsers: `npx playwright install`
3. Run tests: `npm test`
